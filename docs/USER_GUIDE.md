# 🧑‍💻 Руководство Пользователя (MurNet)

Добро пожаловать в **MurNet**! Этот документ написан простыми словами, чтобы любой человек, даже без технического образования, смог понять, зачем нужен этот проект и как им пользоваться.

---

## Что такое MurNet?

**MurNet** — это полностью независимая сеть для общения. 

Обычно, когда вы отправляете сообщение другу в мессенджерах (Telegram, WhatsApp, VK), ваше сообщение летит на огромный сервер, принадлежащий корпорации. Там оно обрабатывается, сохраняется, и только потом отправляется вашему другу. 
**Минусы классического подхода:**
1. Если корпорация захочет (или её заставят), она может заблокировать ваш аккаунт.
2. Сервер может упасть (как бывает при масштабных сбоях), и мессенджер перестанет работать.
3. Вашу переписку могут анализировать алгоритмы.

**MurNet работает иначе:**
Здесь нет серверов. Ваш компьютер напрямую связывается с компьютером вашего друга (или передаёт сообщение через других таких же участников сети по цепочке, словно посылку через доверенных лиц). Эта технология называется **P2P (Peer-to-Peer)**.

---

## Главные фишки сети

1. **🔒 Абсолютная Анонимность**
   Вам не нужно вводить номер телефона, email или придумывать логин. При первом запуске программа генерирует уникальный криптографический ключ. Этот ключ — и есть ваш "паспорт" в сети. Никто не может связать этот ключ с вашей реальной личностью.

2. **🛡️ Невозможность прослушки (Шифрование)**
   Сообщения шифруются на вашем устройстве и расшифровываются только на устройстве получателя. Если кто-то перехватит ваше сообщение по пути (даже если оно прошло через чужой компьютер в MurNet), он увидит лишь случайный "мусор". Прочитать его математически невозможно без ключа вашего собеседника.

3. **⛔ Отсутствие "Рубильника"**
   Так как у MurNet нет центрального сервера, эту сеть невозможно "выключить" или заблокировать провайдером. Пока у вас есть интернет и хотя бы один другой человек с запущенной программой — сеть живет.

4. **📂 Общение и Файлы**
   Вы можете:
   - Писать личные сообщения (Direct Messages).
   - Создавать публичные комнаты по интересам (как каналы, но децентрализованные).
   - Передавать файлы друг другу без ограничений серверов.

## Как начать пользоваться?

Вам не нужно быть программистом. Мы собрали готовые приложения.

### Шаг 1: Загрузка
Перейдите в раздел "Releases" на GitHub или возьмите готовый установочный файл (`.exe` для Windows или бинарник для Mac/Linux) у того, кто скинул вам эту программу.

### Шаг 2: Установка пароля
При первом запуске программа попросит вас придумать **Мастер-пароль**.
⚠️ **ОЧЕНЬ ВАЖНО:** Запомните его! Без него вы никогда не расшифруете свою переписку и не сможете зайти под своим аккаунтом. В MurNet нет кнопки "Забыл пароль" и техподдержки. Если вы потеряли пароль — вы потеряли аккаунт навсегда.

### Шаг 3: Общение
- **Ваш адрес:** В программе вы увидите свой длинный адрес (начинается с набора букв и цифр). Это ваш идентификатор. Дайте его другу, чтобы он смог вам написать.
- **Поиск друзей:** Вы можете "зарегистрировать" себе удобное имя (например, `alice`), чтобы друзья могли находить вас просто по имени, а не по длинному адресу.

### Шаг 4: Безопасность компьютера
Даже если кто-то украдёт ваш ноутбук, он не сможет прочитать ваши переписки или выдать себя за вас. Файл с ключами защищен по военным стандартам (с защитой от подбора паролей мощными видеокартами).

Добро пожаловать в децентрализованное будущее!
